-         Un navigateur (Google Chrome, Mozilla Firefox, etc.) 
-      Ressources extérieures ci-jointes : dossiers contenant les fichiers .sql, .php, .css, etc.

Étapes d’installation
-      Le script de création de la BD ainsi l'insertion d'utilisateurs de tests dans celle-ci se nomme CreationBQ.sql. Il s’agirait simplement d’exécuter ce script dans un environnement SQLServer afin de créer la base de données(1) et de la peupler(2). 
-    Compléter les options de connexion pour les fichiers (présentement vides) infos_connexion.php et pdo_connect.php situés dans le fichier bd_connexion.
 Les options de connexion pour les utilisateurs et la base de données sont placés ailleurs que dans public_html et font ensuite l’objet d’un énoncé “include” dans connexion.php. Comme vos options de connexion à la BD diffèrent des nôtres, veuillez consulter le fichier bd_connexion/infos_connexion.php et bd_connexion/pdo_connect et remplir les espaces nécessaires avec vos propres informations de connexion à la base de données. 
* Placer public_html et bd_connexion sur le serveur.
* Accès à la page de gestion
Un gestionnaire peut avoir accès à la page de gestion(ajout de vidéos) en entrant ses informations de connexion dans la page “form.php”.
